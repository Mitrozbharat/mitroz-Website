What are the required licenses and permissions for using Azure DevOps Services?
Azure DevOps Services are in preview with the following licenses – Azure Professional Services, Azure Advanced Services, and Azure Government Cloud. To use Azure DevOps Services in production, you must acquire an Enterprise license. To use Azure DevOps Services, you will also need the following permissions: Create a resource group, deploy a workload to a resource group, create or edit a job, and manage a pipeline. The permissions vary depending on your license: for instance, the Azure Professional Services license grants permission to create resources and jobs, whereas the Azure Advanced Services license grants permission to create resources and pipelines.
